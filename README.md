# Brevet Booster

Plateforme de révision maths pour le brevet.

## Setup

1. Créer un projet Supabase et exécuter le SQL `supabase-setup.sql`
2. Copier `.env.local.example` en `.env.local` et remplir les clés
3. `npm install`
4. `npm run dev`

## Déploiement sur Vercel

1. Push le code sur GitHub
2. Connecter le repo à Vercel
3. Ajouter les variables d'environnement dans Vercel
4. Déployer !
